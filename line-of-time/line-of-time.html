How to Play

Given a card, guess if it happened before, after, or between the current cards! Put the guesser's name in the current card box. At the end, whoever has the most cards wins! If they get it wrong, make it a new guesser. Click the "+"" button to guess in-between the cards given. The rules follow similar to Chronology.

How to Setup Cards

Each line should have at least the name of the event and the date: Grace Hopper Born, 1906
But, you can also add images by attaching links: Grace Hopper Born, 1906, https://upload.wikimedia.org/wikipedia/commons/9/98/Commodore_Grace_M._Hopper%2C_USN_%28covered%29_head_and_shoulders_crop.jpg
You can save the file you make or load a .txt or .csv file with the same format. Click "hide" when playing so players cannot see the answers, and "unhide" to edit anything yourself.
