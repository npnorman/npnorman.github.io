Some math principles

Given a private key ki and a public key Pi for each person (1 & 2)

The generator g and the prime p, which are g=5, p=23 in the above example.

Person 1 calculates the shared key s by doing (gk2)k1

Note this uses the public key for person 2, P2 = gk2, which assumed hard to find k2.

Person 2 calculates the shared key s by doing (gk1)k2

By multiplying powers, both person 1 and 2 get gk1k2

But if a hacker got P1 and P2, which are public,

gk1 × gk2 = gk1 + k2 ≠ gk1k2
